== 참고해주세요
- 기본적으로 모든 API 는 인증 된 상태의 클라이언트의 사용을 가정합니다. 만약 인증이 불필요한 API 경우 해당 API 제목이나 설명에 기재합니다.
- 빈 값들은 값이 없다는것을 의미합니다. 통일성있는 문서를 위해 빈 값이라도 소제목은 유지합니다.
- 이 문서의 API들은 통합테스트를 온전히 통과한 상태입니다.
